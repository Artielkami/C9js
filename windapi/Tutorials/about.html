About WIND

WIND API is a JavaScript library for manipulating uniformly (independent of the used component) visual, text, cartographic and temporal components.

WIND API allows to abstract mapping providers (OpenLayers v2.13.1, Google Maps v3, Geoportail IGN, OpenStreetMap, Yahoo! Maps, Bing Maps).

For time information, WIND API is based on Simile Timeline v2.3.0 and Chap Links Timeline.
